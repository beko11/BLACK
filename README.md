# Installation

```sh
# Let's install the bot.
: 

cd $HOME

git clone https://github.com/beko11/BLACK.git
cd BLACK

chmod +x black.sh

./black.sh install

./black.sh

# Enter a phone number & confirmation code.

# For Auto Launch:

chmod 777 auto.sh

screen ./auto.sh
```
### One command
To install everything in one command, use:
```sh

:كود واحد
```
